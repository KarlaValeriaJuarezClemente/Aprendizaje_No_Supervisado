# Aprendizaje_No_Supervisado

Este proyecto implementa un análisis de aprendizaje no supervisado para estudiar la relación entre calidad de sueño y salud mental en estudiantes universitarios.
Los algoritmos utilizados son K-means, Mean Shift, Hierarchical clustering y DBSCAN.

# Requisitos Previos

Para ejecutar el proyecto, se recomienda configurar un entorno virtual.

1. Clonar el repositorio:

cd Aprendizaje_No_Supervisado

2. Crear el entorno virtual: Navega a la carpeta raíz de tu proyecto en la terminal y ejecuta:
   py -m venv venv

3. Activar el entorno virtual:
   - En Windows:
     .\venv\Scripts\activate
   - En macOS/Linux
     source venv/bin/activate
