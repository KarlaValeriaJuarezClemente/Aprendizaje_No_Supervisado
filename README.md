# Calidad de Sueño y Salud Mental - Aprendizaje No Supervisado

Este proyecto implementa un análisis de aprendizaje no supervisado para estudiar la relación entre la calidad del sueño y la salud mental en estudiantes universitarios. Los algoritmos utilizados son K-means, Mean Shift, Hierarchical clustering y DBSCAN.

---

## Requisitos Previos

Para ejecutar el proyecto, se recomienda configurar un entorno virtual.

### 1. Clonar el repositorio:
```bash
git clone https://github.com/KarlaValeriaJuarezClemente/Aprendizaje_No_Supervisado.git
cd Aprendizaje_No_Supervisado
```

### 2. Crear el entorno virtual:
Navega a la carpeta raíz de tu proyecto en la terminal y ejecuta:

```bash
py -m venv venv
```

### 3. Activar el entorno virtual:

- **En Windows:**
```bash
.\venv\Scripts\activate
```

- **En macOS / Linux:**
```bash
source venv/bin/activate
```

### 4. Instalar las dependencias:
Una vez que el entorno esté activado, instala las bibliotecas necesarias con el siguiente comando:

```bash
pip install -r requirements.txt
```

### 5. Ejecuta el script principal:

```bash
py AprendizajeNoSupervisado.py
```
